# 研究项目图片文件夹

## 📸 如何替换图片

### 快速步骤：

1. **准备图片**
   - 建议尺寸：800x500 像素
   - 支持格式：.jpg, .png, .svg, .gif
   - 文件名使用英文和连字符（例如：`my-project.jpg`）

2. **上传图片**
   - 将图片文件放到这个文件夹（`assets/images/research/`）
   - 可以直接替换现有的 SVG 文件，或者添加新文件

3. **更新配置**
   - 打开 `_data/research.yml`
   - 找到对应项目的 `image:` 行
   - 修改为：`image: "research/your-image-name.jpg"`

### 免费图片资源：

- **Unsplash**：https://unsplash.com/ （高质量免费图片）
- **Pexels**：https://www.pexels.com/ （免费商用图片）
- **Pixabay**：https://pixabay.com/ （免费图片和矢量图）

### 图片建议：

✅ **适合的图片类型**
- 科技相关的抽象图片
- 数据可视化
- 健康、医疗相关图片
- 人际交流场景
- 简洁、专业的设计

❌ **避免的图片类型**
- 过于花哨或杂乱的图片
- 版权不明的图片
- 含有大量文字的图片
- 像素模糊的图片
